# Setup Guide

Complete setup instructions for Debian security hardening.

## Prerequisites

- Debian 13 (Trixie) or Debian 12 (Bookworm)
- Minimum 2GB RAM
- 500MB free disk space
- Sudo/root access
- YubiKey for SSH authentication

## Installation

1. Clone repository
2. Review scripts
3. Run hardening script
4. Test SSH with YubiKey
5. Restart SSH service
6. Verify all security features

See main README for detailed steps.
